
# bootstragular

"bootstragular" is a demonstration project using Bootstrap and AngularJS with build automation by Grunt.

## installation

```bash

    git clone git@github.com:simon-wenmouth/bootstragular.git

    cd bootstragular

    npm install

    bower install

    grunt

```
